這是一個專為寵物居家照護開發的數位工具。
透過解析 Google 表單的回條文字，自動生成具有品牌權威感且排版精美的「案場執行攻略 PDF」。

## ✨ 功能亮點
- **全自動解析**：一鍵將雜亂的表單回條轉化為模組化指令。
- **職人級排版**：莫蘭迪紫視覺與 A4 雙欄設計，適合行動裝置閱讀與列印。
- **品牌一致化**：鑲嵌專屬黏土貓插圖與社群 QR Code。
- **數據隱私**：所有運算均在瀏覽器端完成，不儲存任何個資。

## 🚀 使用流程
1. **複製**：從 Gmail 或 Google Form 複製完整的客戶回條文字。
2. **貼上**：開啟本工具網頁，將文字貼入輸入框。
3. **生成**：點擊「生成穩定版攻略本」。
4. **保存**：點擊「存為 PDF」，作為進場服務的執行指南。
